# SpokeStack

Agent-native business infrastructure.

Specialized AI agents that manage your tasks, projects, briefs, and orders — learning how your business works and getting smarter every week.

## Get Started

\`\`\`bash
npx spokestack init
\`\`\`

Or send a WhatsApp voice note: [spokestack.dev/start](https://spokestack.dev/start)

## What You Get

- **Tasks Agent** — manages tasks, assignments, and to-dos (Free)
- **Projects Agent** — handles project planning, workflows, and timelines (Starter, $29/mo)
- **Briefs Agent** — orchestrates creative/strategic briefs with artifact generation (Pro, $59/mo)
- **Orders Agent** — manages customers, orders, and invoicing (Business, $149/mo)

Every agent reads the same shared context graph. The more you use any agent, the smarter they all get.
